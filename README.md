# ParkPulse 🚗

ParkPulse is a modern campus parking web app designed to help students find parking efficiently and avoid wasting time searching for spots.

## 🚀 Features
- Real-time parking concept
- Clean mobile-friendly UI
- Campus parking map
- Smart parking insights

## 🛠 Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## 📌 Why I Built This
I created ParkPulse during a buildathon to solve a real problem students face daily — finding parking on campus. This project showcases my ability to build clean, responsive interfaces and structure a real-world application.
